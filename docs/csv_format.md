# Supplier CSV Specification

The system generates a CSV with the following columns for IndiaMART suppliers:

1.  **Order ID**: Internal reference (e.g., `order_Pz123...`)
2.  **Customer Name**: Full name
3.  **Phone**: Customer contact
4.  **Address**: Full address string (quoted)
5.  **City**
6.  **State**
7.  **Zip**: Pincode
8.  **Product SKU**: `SNS-MINI-PK01`
9.  **Qty**: Quantity ordered
10. **Notes**: Special instructions
